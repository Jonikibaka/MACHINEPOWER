# Приложение для предсказания оценки ученика
- Ввод характеристик ученика: возраст, часы учёбы, посещаемость.  
- Ввод оценок по предметам: математика, физика/наука, английский, общий балл.  
- Выбор категориальных признаков: тип школы, наличие дополнительных активностей (да/нет), метод обучения.  
- Отображение предсказанной финальной оценки во всплывающем окне.  
- Киберпанковый переливающийся дизайн интерфейса.
